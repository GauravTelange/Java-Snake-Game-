# Java Snake Game

## 📌 Project Overview
The Java Snake Game is a 2D desktop game developed using Java Swing. The game implements core game mechanics such as snake movement, food generation, collision detection, and score tracking.

## 🚀 Features
- Snake movement in four directions
- Random food generation
- Score tracking
- Collision detection with walls and snake body
- Game over functionality

## 🛠️ Technologies Used

- Java
- Java Swing
- Arrays
- Random Class

## 🎮 Game Mechanics

Snake Movement
- Up
- Down
- Left
- Right

Food Generation
- Food appears randomly on the screen

Collision Detection
- Snake hits wall → Game Over
- Snake hits itself → Game Over

Score System
- Score increases when snake eats food

## ▶️ How to Run the Project

1. Clone the repository
2. Open project in any Java IDE
3. Compile and run the main class
